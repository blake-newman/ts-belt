ts-belt

Fast, modern, and practical utility library for FP in TypeScript.

Getting started
Go to API
👀 provides more readable code, due to the data-first approach·✨ supports TypeScript and Flow·🛡 helps you write safer code with Option and Result data types·🎯 all functions return immutable data (no side-effects)·🌲 tree-shakeable
